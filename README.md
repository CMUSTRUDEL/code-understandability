# code-understandability

R source code for:

```
“Automatically Assessing Code Understandability” Reanalyzed: Combined Metrics Matter.
Asher Trockman, Keenen Cates, Mark Mozina, Tuan Nguyen, Christian Kästner, and Bogdan Vasilescu.
In Proceedings of the International Conference on Mining Software Repositories, MSR, ACM (2018).
```

This study is a reanalysis of the data set provided by:

```
Automatically assessing code understandability: how far are we?.
Simone Scalabrino, Gabriele Bavota, Christopher Vendome, Mario Linares-Vásquez, Denys Poshyvanyk, and Rocco Oliveto.
In Proceedings of the 32nd IEEE/ACM International Conference on Automated Software Engineering, pp. 417-427. IEEE Press, 2017.
```

The data set is included in this repository (`understand.csv`), and also can be found at https://dibt.unimol.it/report/understandability/.
